# Video Summary Template

This template combines useful patterns from:

- Fabric `summarize` and `extract_wisdom`: concise summary, ideas, insights, habits, references, recommendations.
- Armory `youtube-analysis`: content type detection, key concepts, timestamps, technical terms, takeaways.
- OpenClaw video summary skill: note-friendly Markdown sections and metadata.

Use the main `SKILL.md` output format unless a user requests a different shape.
